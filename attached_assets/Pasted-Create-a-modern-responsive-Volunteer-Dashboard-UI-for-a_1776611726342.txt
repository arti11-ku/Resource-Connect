Create a modern, responsive Volunteer Dashboard UI for a web application called "SAHARA – Smart Resource Allocation System".

Design Style:
- Clean, minimal UI with Indian-inspired saffron/orange (#FF7A00) and white theme
- Soft gradients (light orange + white), not heavy colors
- Use Poppins font
- Rounded cards, subtle shadows, smooth hover effects
- Professional dashboard look (like real-world platforms)

- Left: SAHARA logo

2. Sidebar (Left):

- Available Tasks

- Scoreboard

- Settings

- List of assigned tasks
- Status: Pending / In Progress / Completed

E. Volunteer Profile Summary:
- Skills (tags)
- Availability (Day/Night, Weekdays/Weekends)
